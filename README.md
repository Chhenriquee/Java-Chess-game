# Jogo de Xadrez em Java ☕

Essa aplicação trata-se de um jogo de xadrez programado em JavaSE. Todas as pessoas fazem os movimentos conforme o jogo, sendo dividido em duas equipes, branco e preto (WHITE, BLACK).
